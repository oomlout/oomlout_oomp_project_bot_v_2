## 概要

SOPi装置のフロントパネルに装着するLED用の基板。  
OPiなどの3.3Vの場合には510Ωから1kΩ程度の抵抗で良いと感じる。

実装の向きなどは、3Dグラフィックを参照の上、組み立てる。

![BackSide](M912-01.png)
![FrontSide](M912-02.png)
![SideView](M912-03.png)

なお、LEDが赤で表示されているが、色を特定するものではない。
