SparkFun DC/DC Converter Breakout
========================================

![SparkFun DC/DC Converter Breakout](https://cdn.sparkfun.com//assets/parts/2/9/6/1/09370-1.jpg)

[*SparkFun DC/DC Converter Breakout (BOB-09370)*](https://www.sparkfun.com/products/9370)

This is a breakout board for our non-isolated, 6A DC-to-DC Converter Module. 
The module can convert any DC voltage between 4.5-14VDC to a selectable voltage between 0.59 and 5.5VDC.
