RJ45 Breakout
=============

[![RJ45 Breakout](https://cdn.sparkfun.com//assets/parts/4/7/7/00716-06.jpg)
*SparkFun RJ45 Breakout (BOB-00716)*](https://www.sparkfun.com/products/716)

This board brings out the RJ45 pins to in-line 0.1" headers.
