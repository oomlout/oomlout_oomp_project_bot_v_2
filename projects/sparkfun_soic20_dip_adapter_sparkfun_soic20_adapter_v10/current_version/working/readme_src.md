SOIC20 DIP Adapter
==================

[![SOIC20 to DIP Adapter](https://cdn.sparkfun.com//assets/parts/2/8/1/00495-01.jpg)
*SparkFun SOIC to DIP Adapter 20-Pin (BOB-00495)*](https://www.sparkfun.com/products/495)

This pcb breaks out standard SOIC 20 pin packages to 0.1" headers.
