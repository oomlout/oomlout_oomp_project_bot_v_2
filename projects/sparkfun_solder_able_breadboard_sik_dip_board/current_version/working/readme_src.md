Solder-able Breadboard
=======================

[![Solder-able Breadboard](https://dlnmh9ip6v2uc.cloudfront.net/images/products/1/2/0/7/0/12070-01_medium.jpg) 
*Solder-able Breadboard(PRT-12070)*](https://www.sparkfun.com/products/12070)
