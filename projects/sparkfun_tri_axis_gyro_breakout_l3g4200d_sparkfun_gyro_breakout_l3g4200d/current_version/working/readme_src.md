SparkFun Tri-Axis Gyro Breakout - L3G4200D
========================================

![SparkFun Tri-Axis Gyro Breakout - L3G4200D](https://cdn.sparkfun.com//assets/parts/5/2/6/6/10612-02.jpg)

[*SparkFun Tri-Axis Gyro Breakout - L3G4200D (SEN-10612)*](https://www.sparkfun.com/products/10612)

This is a breakout board for the L3G4200D low-power three-axis angular rate sensor. 
The L3G4200D is a MEMS motion sensor and has a full scale of ±250/±500/±2000 dps and is capable of measuring rates with a user-selectable bandwidth.
