SparkFun TRRS 3.5mm Jack Breakout
========================================

![SparkFun TRRS 3.5mm Jack Breakout](https://cdn.sparkfun.com//assets/parts/7/5/4/6/11570-01.jpg)

[*SparkFun TRRS 3.5mm Jack Breakout (BOB-11570)*](https://www.sparkfun.com/products/11570)

This breakout board makes it easy to add a TRRS jack to your prototype or project by breaking out each conductor to a standard 0.1" spaced header.
Repository Contents
-------------------

* **/Hardware** - Eagle design files (.brd, .sch)
* **/Production** - Production panel files (.brd)
