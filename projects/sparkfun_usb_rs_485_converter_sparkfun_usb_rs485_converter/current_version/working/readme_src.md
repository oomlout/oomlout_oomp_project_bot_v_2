SparkFun USB to RS-485 Converter
========================================

![SparkFun USB to RS-485 Converter](https://cdn.sparkfun.com//assets/parts/3/8/7/8/09822-01b.jpg)

[*SparkFun USB to RS-485 Converter (BOB-09822)*](https://www.sparkfun.com/products/9822)

This breakout board pairs an SP3485 RS-485 transceiver with an FT232RL USB UART IC to convert a USB stream to RS-485.

Repository Contents
-------------------

* **/Hardware** - Eagle design files (.brd, .sch)
* **/Production** - Production panel files (.brd)
